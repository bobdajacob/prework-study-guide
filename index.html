HTML

  • Title sets the title of the webpage inside the head element
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Button is used for clickable button on a webpage
  • Class attribute is used to define HTML elements

CSS

  • CSS is used for styling HTML documents using many different properties
  • You can style any property with color, background-color, width, height, etc.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Repositoyry is a storage where you can put your project
  • Branches allows you to work on different part of a project
  • Merging is combining the branches

JavaScript

  • JavaScript manipulates the content of a webpage
  • Essential for building complex web applications
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
